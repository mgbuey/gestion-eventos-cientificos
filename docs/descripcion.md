# Funcionalidad básica del proyecto

La aplicación permite la creación y edición de eventos, la gestión de ponentes y asistentes, el registro de contribuciones científicas (charlas, pósters, sesiones), así como la asignación de horarios y salas.

Además, incluiye funcionalidades para la inscripción de participantes, el envío de notificaciones y la consulta del programa por parte de los usuarios.
